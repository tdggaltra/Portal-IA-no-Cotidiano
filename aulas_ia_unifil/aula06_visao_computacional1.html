👁️ O que é Visão Computacional?

Como a IA “vê”

Visão Computacional ensina máquinas a interpretar e entender imagens e vídeos — do reconhecimento facial ao diagnóstico médico.

🔍 Tarefas Principais

  • Classificação: “é um gato ou cachorro?”
  • Detecção: onde estão os objetos na imagem?
  • Segmentação: delimitar contornos pixel a pixel
  • OCR: reconhecer texto em imagens
  • Pose estimation: detectar posição do corpo
  • Reconhecimento facial: identificar pessoas

💻 Aplicações no dia a dia

  • Desbloqueio por reconhecimento facial (celular)
  • Filtros de realidade aumentada (Instagram, Snapchat)
  • Carros autônomos identificando objetos
  • Caixas de supermercado sem atendente
  • Controle de qualidade em fábricas
  • Diagnóstico médico por imagens de raio-X

🔧 Teachable Machine

Crie seu próprio modelo de visão sem código

🌻

Google Teachable Machine

teachablemachine.withgoogle.com — totalmente gratuito e no browser

✅ O que dá para fazer

  • Treinar classificador de imagens, sons ou poses
  • Coletar exemplos com a webcam em tempo real
  • Testar o modelo na hora, ainda no browser
  • Exportar para TensorFlow.js, TFLite ou Keras
  • Integrar em sites e apps facilmente

🎮 Casos de uso práticos

  • Detectar se aluno está com máscara ou não
  • Identificar gestos de mão para controlar algo
  • Reconhecer frutas ou objetos específicos
  • Classificar qualidade de produtos na linha de produção
  • Criar jogo controlado por expressão facial

🌻 Limite importante

O Teachable Machine é ótimo para prototipação rápida, mas não para produção. Para soluções robustas, use YOLO, TensorFlow ou serviços cloud como Google Cloud Vision ou AWS Rekognition.

📸 Ferramentas de Visão com IA

Do gratuito ao profissional

🔊 Google Cloud Vision

  • API de visão computacional
  • Detecta objetos, rostos, texto, logos
  • Rótulos automáticos em imagens
  • Moderação de conteúdo
  • 1.000 requisições gratuitas/mês

📋 GPT-4o Vision

  • Analisa qualquer imagem enviada ao ChatGPT
  • Responde perguntas sobre o conteúdo
  • Descreve cenas, objetos, textos
  • Interpreta gráficos, tabelas, equações
  • Totalmente disponível no plano gratuito!

📶 Roboflow

  • Plataforma para treinar detectores de objetos
  • Dataset annotation integrado
  • Treinamento na nuvem sem configuração
  • Deploy em API ou dispositivos edge
  • Plano gratuito para projetos acadêmicos

💡 Dica prática

Para começar hoje: use o ChatGPT enviando uma imagem e fazendo perguntas sobre ela. É gratuito e funciona incrivelmente bem para análise rápida de imagens.

🔬 Laboratório — Aula 6

Treinando nosso próprio modelo de visão

🌻

Hora de Treinar!

Cada grupo vai criar um modelo de classificação de imagens

📋 Passo a passo

  • Acesse teachablemachine.withgoogle.com
  • Crie projeto “Image Project” — Standard model
  • Crie 3 classes com pelo menos 40 fotos cada
  • Use a webcam para coletar os exemplos
  • Treine o modelo e teste em tempo real
  • Exporte e integre em uma página HTML simples!

🎯 Sugestões de Projetos

  • Linguagem de sinais: A, B, C em Libras
  • Emoções faciais: feliz, triste, surpreso
  • Gestos: polegar cima, baixo, certo com dedos
  • Objetos de mesa: caneta, télefone, livro
  • Livre: o que a equipe escolher!
