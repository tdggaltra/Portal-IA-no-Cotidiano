🤔 Por que prompts importam?

Garbage in, garbage out

"A qualidade da resposta da IA é diretamente proporcional à qualidade do seu prompt."

❌ Prompt fraco

  • “Me fala sobre marketing”
  • “Escreve um código pra mim”
  • “Analisa isso” (sem contexto)
  • “Melhora meu texto”

✅ Prompt forte — Exemplo

  • Papel: “Você é consultor de marketing digital B2B”
  • Contexto: startup de SaaS, público de RH, orçamento R$2.000/mês
  • Tarefa: crie 5 estratégias de conteúdo detalhadas
  • Formato: lista numerada com justificativa para cada uma

🧩 Anatomia de um Bom Prompt

Os 5 elementos que fazem a diferença

1. 🎪 Papel (Role)

Diga à IA quem ela é:

“Você é um professor universitário especialista em redes...”

2. 📝 Contexto

Informe a situação:

“...alunos de 2º ano de TI que já sabem Python...”

3. 🎯 Tarefa

Diga o que quer:

“...crie um exercício sobre TCP/IP com 3 questões...”

4. 📐 Formato

Especifique a saída:

“...Markdown, nível intermediário, máx 300 palavras”

✅ Prompt Completo Exemplo

"Você é um professor universitário especialista em redes. Estou preparando aula para alunos de 2º ano de TI. Crie um exercício prático sobre TCP/IP com 3 questões de múltipla escolha e gabarito. Formato: Markdown. Nível: intermediário. Máx 300 palavras."

🚀 Técnicas de Prompting

Zero-shot, few-shot e chain-of-thought

⚡ Zero-Shot

  • Instrução direta sem exemplos
  • Funciona para tarefas simples e bem definidas
  • Mais rápido de escrever
  • “Classifique este e-mail como urgente ou não”

📚 Few-Shot

  • Fornece 2–5 exemplos antes da tarefa
  • Ensina o padrão desejado
  • Muito eficaz para transformações de texto
  • “Urgente: servidor caiu → sim | reunião amanhã → não | Classifique: bug em produção”

🧠 Chain-of-Thought

  • Pede para a IA “pensar em voz alta”
  • Melhora resultados em raciocínio lógico
  • Ideal para problemas matemáticos e análises
  • “Resolva passo a passo, explicando seu raciocínio”

🔑 Dica de Ouro

Prompting é iterativo! Comece simples, veja o resultado, ajuste, repita. Trate como uma conversa — use o contexto anterior para refinar. Nunca descarte uma boa conversa!

💡 Técnicas Avançadas

Prompts que entregam resultados profissionais

🎯 Delimitadores e Estrutura

  • Use ### para separar partes do prompt
  • Especifique o público-alvo: “para leigos”, “nível técnico”
  • Peça tom: formal, descontraído, persuasivo
  • Use negativos: “sem jargões”, “sem emojis”
  • Limite: “máximo 200 palavras”

🔄 Prompts de Refinamento

  • “Refaça mais conciso (máx. 100 palavras)”
  • “Adicione dados e estatísticas a cada ponto”
  • “Torne mais persuasivo para público de RH”
  • “Converta para formato de tabela”
  • “Dê 3 alternativas para o parágrafo 2”

🎪 Persona Especialista

“Aja como um advogado especialista em LGPD e revise este contrato...”

🔮 Simulação de Cenários

“Simule 3 objeções que um investidor faria a esta proposta...”

🧪 Revisão Crítica

“Seja meu crítico mais severo e aponte falhas neste argumento...”

💻 Prompts para Código

IA como parceiro de programação

⌨️

IA como par de programação

Gerar, explicar, depurar e documentar código

🔧 Tipos de tarefas de código

  • Gerar: “Escreva uma função Python que...”
  • Explicar: “Explique linha por linha o que faz...”
  • Depurar: “Esse código dá [erro X], corrija...”
  • Refatorar: “Melhore a legibilidade mantendo a lógica...”
  • Converter: “Converta de JavaScript para TypeScript...”
  • Documentar: “Adicione docstrings a todas as funções...”

📋 Prompt de Código Ideal

  • Especifique a linguagem e versão
  • Descreva o input e output esperado
  • Mencione restrições (sem libs externas, etc.)
  • Peça comentários no código
  • Inclua exemplo de uso no prompt
🔎 Sempre teste o código gerado! A IA pode gerar código que parece certo mas tem bugs sutis.
