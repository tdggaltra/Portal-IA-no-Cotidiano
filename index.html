Como publicar no GitHub Pages

01
Crie um repositório público no GitHub com o nome seu-usuario.github.io ou qualquer nome.
02
Suba este index.html na raiz + a pasta aulas_ia_unifil/ + os arquivos de atividade.
03
Em Settings → Pages, selecione a branch main e pasta / (root). Salve.
04
Aguarde ~2 minutos e acesse seu-usuario.github.io/nome-repo. Tudo estático, sem custo!
05
Para novas aulas, basta fazer git push — o GitHub Pages atualiza automaticamente em segundos.
Dica
Adicione um arquivo .nojekyll vazio na raiz para evitar que o GitHub processe desnecessariamente os arquivos HTML.
